Vremenska prognoza, plugin koji prikazuje vremensku prognozu za odreÄ‘eni grad. Potreban API kljuc, OpenWeatherMap.

== Installation ==

Install Vremenska Prognoza plugin to your blog, activate it, and then enter your OpenWeatherMap API key.
To show forecast you need to use shortcode [prognoza].

== Description ===

This plugin uses OpenWeatherMap 3rd party service. If you want to get weather forecast for your city you need to get API key from OpenWeatherMap, otherwise plugin
will not work.

Link to this service: https://openweathermap.org
Privacy policy of OpenWeatherMap: https://openweather.co.uk/privacy-policy
